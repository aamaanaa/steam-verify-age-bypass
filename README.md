```text
Bypasses the annoying age prompt on the Steam Store page.
---

USECASE
Due to legal requirements, steam is forced to always request your age when browsing games with a PEGI rating of 16 or higher. This disables that nonsense once and for all.

DOWNLOAD
Either download from the releases, or compile the binary yourself.

USAGE
For Linux:
An example of how to use the precompiled binary in releases:
./bypass.elf --file  /home/urusername/.steam/steam/config/htmlcache/Cookies

For Windows:
The same, but using the windos executable.

SUPPORT
Only Linux/Windows is supported. Windows support is best efford only. All else will not be supported.

ISSUES
THis is a personal project provided as is that it may or may not be usefull for you.
Therefore, no issue tracking, no support.

LICENSE
None. do whatever you want with it.

```
